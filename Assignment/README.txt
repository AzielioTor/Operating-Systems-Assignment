You may run this program with defult input parameters by running it.
(Default parameters are 25 seconds, 5 producers threads, 5 consumer threads)

Or you may run the program by passing in 3 parameters, the first is how long
you want the program to run (In milliseconds), the second is how many producer
threads you want, the third is how many consumer threads you want.
If any of these inputs arent of valid the default parameters will be used.
(Default parameters are 25 seconds, 5 producers threads, 5 consumer threads)
